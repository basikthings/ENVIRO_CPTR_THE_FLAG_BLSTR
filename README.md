# ECE520 Final Project

**Introduction**

For my final project for ECE520, I will be building a capture the flag platformer game. 

**Agents**
- Main Player 
    - Mouse Click Control
    - Life Bar decreases weight. More knock back.
    - Melee Attack 
- Ghosts  
    - Melee Attack 
    - Life Bar 
    - Spawns from Base 
    - Protects Flag 
- Tower/Flag 
    - Life Bar 
    - Grab Flag 
    - Return to Base
